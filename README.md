# EasyWheels
A Python-based dealership management system designed to handle inventory and customer transactions efficiently.

## Project Description
EasyWheels is a modular Python application that helps manage car inventory and dealership operations. It utilizes Object-Oriented Programming (OOP) to organize code into distinct entities for better maintenance and scalability.

## Key Features
* *Car Management*: Track vehicle details (VIN, Make, Model, Price).
* *Dealership Operations*: Add, remove, and view inventory.
* *Modular Design*: Separated into car.py, customer.py, dealership.py, transaction.py, and main.py for clean code organization.
* *Data Persistence*: Includes basic File I/O for inventory management.
